The Segno <ds-disclosure> element toggles the visibility of a Control Element to show/hide content.

Display
block
Parent
n/a
Children
n/a
Control Element

A Control Element should have a boolean, open property that reflects to a boolean, open HTML attribute.

  • The open attribute is use to initialize the Disclosure's state.
  • The open property is inverted when the Disclosure is triggered.

The <ds-reveal> element meets the requirements to be a Control Element for a Disclosure.

{% endblock %} {% block attributes %}
aria-controls (String)
ID of an HTML element (Control Element) to associate behavior. This is required to wire up display functionality between elements.
aria-expanded
Automatically initialized to the same value as its Control Element. When this attribute is updated, the Control Element is also updated.
role*
Automatically set to "button", if not explicitly defined in markup.
tabindex*
Automatically set to 0 (placing it in the tab order), if not explicitly defined in markup.
{% endblock %} {% block properties %}
expanded (Boolean)
Mirrors the aria-expanded attribute.
  • Set to true to expand its Control Element.
  • Set to false to collapse its Control Element.
