$generateId() → {String}

The string generated from this method is not guaranteed to be 100% unique. However, it seems to be good enough for use within element IDs.

Uses a pseudo-random number generator to generate a "unique" 8-character string for use as all or part of a unique identifier in an HTML document.

$upgradeProperty(propertyName)

Ensures that properties set on an element before it's upgraded do not overrule the custom element's property setter after it's upgraded.

Best used in the connectedCallback() lifecycle method (when the element is attached to the DOM).
