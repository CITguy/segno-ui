The Grid provides CSS classes using flexbox to arrange contents in a single direction (vertically or horizontally).

Rows

Rows are containers that allow you to arrange contents horizontally along the positive X-axis (left to right).

Column Spans

Column span modifiers will adjust the width of a Column container to a predefined percentage of the Row width.

Start with the .dsCol-{N}-xs modifier (1 ≤ N ≤ 12) to create columns that are a fraction of a 12-column row.

See Responsive Options below for more details.

Ordering Columns

Start with the .dsOrder-{N}-xs class (1 ≤ N ≤ 12) to reorder columns.

See Responsive Options below for more details.

The columns below are in DOM order of 2-4-1-3, but the .#{N} classes reordered them in the correct numerical order.

#2

#4

#1

#3

<div class="dsRow">
  <div class="dsCol dsOrder-2-xs"><p>#2</p></div>
  <div class="dsCol dsOrder-4-xs"><p>#4</p></div>
  <div class="dsCol dsOrder-1-xs"><p>#1</p></div>
  <div class="dsCol dsOrder-3-xs"><p>#3</p></div>
</div>

Responsive Options

All column modifiers documented above support different breakpoint modifiers.

Each responsive class corresponds to the minimum breakpoint needed to achieve the desired result. Larger breakpoints override smaller ones.

CSS Class Minimum Breakpoint Overrides Overridden By
Span Offset Order
.dsCol-{N}-xs .dsOffset-{N}-xs .dsOrder-{N}-xs 0em (0px) n/a All below
.dsCol-{N}-sm .dsOffset-{N}-sm .dsOrder-{N}-sm 40em (~640px) All above All below
.dsCol-{N}-md .dsOffset-{N}-md .dsOrder-{N}-md 64em (~1024px) All above All below
.dsCol-{N}-lg .dsOffset-{N}-lg .dsOrder-{N}-lg 75em (~1200px) All above All below
.dsCol-{N}-xl .dsOffset-{N}-xl .dsOrder-{N}-xl 90em (~1440px) All below n/a

Column Wrapping

Column containers will wrap to a new Row if the column count exceeds 12 or if there's not enough room for a column on the same row. Both column span and column offset contribute toward the total column count.

The following Columns should self-arrange into 2 columns on extra small screens, 3 columns on small screens, 4 columns on medium screens, and 6 columns on larger screens.
