<ds-tablist>

The Segno <ds-tablist> element is a child container of a <ds-tabset> that groups together all of your <ds-tab> elements.

Usage

Display
block
Parent
<ds-tabset>
Children
<ds-tab> 
<ds-tablist
  [role="{String}"]>

  ...
</ds-tablist>

Attributes

Attributes allow you to configure the element in a declarative manner.

aria-orientation*
Automatically set by <ds-tabset> when setting the tab-side attribute or tabSide property.
role*
Automatically set to "tablist", if not explicitly defined in markup.

Properties

Properties allow you to configure the element in an imperative manner.

This element does not have any exposed properties.

Functions

Functions alllow you to trigger complex behavior in an element.

This element does not have any exposed functions.

Compatibility

IE11 Edge Chrome Safari Firefox
