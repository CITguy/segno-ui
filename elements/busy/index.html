<ds-busy>

Usage

The Segno <ds-busy> element communicates that a process of unknown duration is taking place.

Display
inline-block
Parent
n/a
Children
None
<ds-busy [paused]></ds-busy>

Examples

Attributes

Attributes allow you to configure the element in a declarative manner.

paused
Pauses animation, when present.

Properties

Properties allow you to configure the element in an imperative manner.

paused (Boolean [false])
Pauses animation, when set to true.

Functions

Functions alllow you to trigger complex behavior in an element.

This element does not have any exposed functions.

Compatibility

IE11 Edge Chrome Safari Firefox
