Usage

<ds-busy [paused]></ds-busy>

Attributes

Attributes allow you to configure the component in a declarative manner.

paused
Pauses animation, when present.

Properties

Properties allow you to configure the component in an imperative manner.

paused (Boolean [false])
Pauses animation, when set to true.

Functions

Functions alllow you to trigger complex behavior in a component.

This component does not have any exposed functions.
