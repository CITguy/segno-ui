<ds-tab>

The <ds-tab> element is the interactive label to an associated <ds-tabpanel> element.

Usage

Display
block
Parent
<ds-tablist>
Children
n/a
<ds-tab
  [current]
  [id="{String}"]>

  ...
</ds-tab>

Attributes

Attributes allow you to configure the element in a declarative manner.

aria-controls*
Automatically set by <ds-tabset> when attached to the DOM.
aria-selected*
Automatically set as the current attribute/property changes.
current*
Automatically applied by <ds-tabset> to identify the current Tab in a Tablist.
id (String)
Used by <ds-tabset> to wire up a Tab to its Tabpanel using the aria-controls and aria-labeledby attributes. If you do not define an id attribute, one will be automatically generated.
role*
Automatically set to "tab", if not explicitly defined in markup.

Properties

Properties allow you to configure the element in an imperative manner.

current (Boolean [false])
  • Set to true to set current Tab state.
  • Set to false to remove current Tab state.

Functions

Functions alllow you to trigger complex behavior in an element.

This element does not have any exposed functions.

Compatibility

IE11 Edge Chrome Safari Firefox
