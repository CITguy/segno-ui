<ds-reveal>

Usage

The Segno <ds-reveal> element controls the visibility of a section of content.

Display
block (when open)
none (when closed)
Parent
n/a
Children
n/a
<ds-reveal [open]>
  ...
</ds-reveal>

Examples

Why'd the cookie cry?

Toggle Answer

Because his father was a wafer so long!

 
<p>Why'd the cookie cry?</p>
<ds-disclosure aria-controls="demoReveal">
  <small>Toggle Answer</small>
</ds-disclosure>
<ds-reveal id="demoReveal">
  <p>...</p>
</ds-reveal>

Attributes

Attributes allow you to configure the element in a declarative manner.

open
When present, contents are visible.

Properties

Properties allow you to configure the element in an imperative manner.

open (Boolean [false])
  • Set to true to expand the Reveal
  • Set to false to collapse the Reveal

Functions

Functions alllow you to trigger complex behavior in an element.

This element does not have any exposed functions.

Compatibility

IE11 Edge Chrome Safari Firefox
