Attributes

Attributes allow you to configure the element in a declarative manner.

current-tab (Number)
(default: 0) Zero-based index of the currently active Tab + Tabpanel
tab-side (String: 'top' | 'right' | 'bottom' | 'left')
(default: 'top') Position of the tablist in relation to tab contents.

Properties

Properties allow you to configure the element in an imperative manner.

currentTab (Number)
Zero-based index of the currently active Tab + Tabpanel
tabSide (String)
Position of the tablist in relation to tab contents.

Functions

Functions alllow you to trigger complex behavior in an element.

This element does not have any exposed functions.
