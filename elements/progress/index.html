Progress

Demo

Usage

<ds-progress
  [value="{Number}"]
  [indeterminate]>
</ds-progress>

Attributes

Attributes allow you to configure the component in a declarative manner.

value (Number)
Progress percentage (0-100)
indeterminate (Boolean)
When present, set the element to an indeterminate state.

Properties

Properties allow you to configure the component in an imperative manner.

value (Number)
(default: 0) progress percentage (0-100)
indeterminate (Boolean)
(default: false) When true, set the element to an indeterminate state.

Functions

Functions alllow you to trigger complex behavior in a component.

This component does not have any exposed functions.

Compatibility

IE11 Edge Chrome Safari Firefox
  • Be aware that dynamic updates to the color property may or may not update the indeterminate stripes in older browsers.
