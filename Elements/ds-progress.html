<ds-progress>

Usage

The Segno <ds-progress> element communicates completion progress for a task.

DOM API
DSProgressElement
Display
block
Parent
n/a
Children
None
<ds-progress
  [value="{Number}"]
  [indeterminate]>
</ds-progress>

Examples

Attributes

Attributes allow you to configure the element in a declarative manner.

value {Number=0}
Progress percentage (0-100)
indeterminate {Boolean=false}
When present, set the element to an indeterminate state.
