<ds-icon>

Usage

The Segno <ds-icon> element renders one of several SVG icons.

DOM API
DSIconElement
Display
inline-block
Parent
n/a
Children
None - Children will be replaced when the element upgrades.
<ds-icon type="{String}"></ds-icon>

Attributes

Attributes allow you to configure the element in a declarative manner.

type
Set the value to one of the values documented above.
