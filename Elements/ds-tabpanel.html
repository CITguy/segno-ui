<ds-tabpanel>

Usage

The Segno <ds-tabpanel> element is a child container of a <ds-tabcontent> that contains the contents for a specific tab.

Attributes

Attributes allow you to configure the element in a declarative manner.

auto area-expanded {String}
Automatically set as the open attribute/property changes.
id {String}
Used by <ds-tabset> to wire up a Tab to its Tabpanel using the aria-controls and aria-labeledby attributes. If you do not define an id attribute, one will be automatically generated.
open {Boolean=false}
When present, contents are visible.
auto role {String="tabpanel"}
Automatically set to "tabpanel", if not explicitly defined in markup.
