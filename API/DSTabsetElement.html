Properties

currentTab {Number=0} [RW]
Zero-based index of the currently active Tab + Tabpanel
tabs {Array} [RO]
Array of all <ds-tab> elements in the tabset.
tabpanels {Array} [RO]
Array of all <ds-tabpanels> elements in the tabset.
tabSide {String="top"} [RW]
Position of the tablist in relation to tab contents.

Methods

None specified; inherits methods from its parent.
