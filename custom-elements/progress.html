<ds-progress>

Usage

The Segno <ds-progress> element communicates completion progress for a task.

Display
block
Parent
n/a
Children
None
<ds-progress
  [value="{Number}"]
  [indeterminate]>
</ds-progress>

Examples

Attributes

Attributes allow you to configure the element in a declarative manner.

value (Number)
Progress percentage (0-100)
indeterminate (Boolean)
When present, set the element to an indeterminate state.

Properties

Properties allow you to configure the element in an imperative manner.

value (Number)
(default: 0) progress percentage (0-100)
indeterminate (Boolean)
(default: false) When true, set the element to an indeterminate state.

Functions

Functions alllow you to trigger complex behavior in an element.

This element does not have any exposed functions.

Compatibility

IE11 Edge Chrome Safari Firefox
  • Be aware that dynamic updates to the color property may or may not update the indeterminate stripes in older browsers.
